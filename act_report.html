This act report includes the summary of the Data Analysis process that was taken for the data wrangling project undertaken.¶

For this project, I worked with three datasets.

Udacity provided the first dataset as a csv file named twitter_archive_enhanced.csv. It contained basic information of about 2356 tweets and it was downloaded manually.

The second dataset was a tsv file named image_prediction.tsv, which was hosted on the udacity server and had to be programmatically downloaded. It contained 2075 image predictions made by a neural network that can identify dog breeds.

The third and final dataset was scrapped from the twitter API using python Tweepys Library. This third dataset contained information like the retweet count, favorite count, followers count and friends count for each tweet with 2327 tweets saved into a file named "tweet_json_text".

During the data aassessment stage, 11 data quality issues and 4 data tidiness issues were found respectively in the three datasets. A variety of Pandas methods were used to assess and clean these issues found in the datasets.

Some insights and visualizations from the three datasets that were merged into a master dataset named twitter_archive_master.csv.¶

Insights¶

  • retweet count had a minimum value of 11, mean is 2241, and the maximum retweet count is 70662

  • Approximately 32% of the all dogs in the dataset had no names

  • Image number 1 had the highest frequency among all images in the neural network's image predictions

  • The merged dataset has 1986 rows and 21 columns, all the rows except for the stage column which is a combined column for all the dog stages are completely filled with no missing values.
